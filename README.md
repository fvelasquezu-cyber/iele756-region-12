# iele756-region-12
Integrantes: Felipe Velásquez, Matías Moraga

Región: Región de Magallanes y de la Antártica Chilena (XII)

Este proyecto consiste en el análisis de datos sobre salud pública en la Región de Magallanes, Chile, utilizando bases de datos nacionales actualizadas al 2024. A través del uso de Python en Google Colab.

Enfermedades de Notificación Obligatoria (ENO) y los Egresos Hospitalarios (GRD/DEIS).
